# Frequently Asked Questions (FAQ)

## GDevelop
**What is GDevelop?**  
GDevelop is a free and open-source game development platform designed for everyone, allowing users to easily create games without any programming knowledge.

**Is GDevelop suitable for beginners?**  
Yes, GDevelop is particularly beginner-friendly and provides a visual programming model that makes it easy to get started.

## Technical Requirements
**What are the system requirements for GDevelop?**  
GDevelop is compatible with Windows, macOS, and Linux, with recommended specifications including at least 4GB of RAM and a modern processor.

**Do I need to install any software?**  
You can use GDevelop in a web browser without any installation, or download the desktop version for offline usage.
