# First Project in GDevelop

## Steps to Create a New Project
1. Open GDevelop.
2. Click on "Create a New Project".
3. Choose a template or start from scratch.
4. Name your project and click "Create".

## Understanding Scenes
- Each project consists of one or more scenes.
- Scenes are individual screens within your game.

## Adding Objects
1. Go to the "Objects" section.
2. Click on "Add an Object".
3. Choose the type of object (sprite, text, etc.).
4. Import graphics or draw your object.

## Adding Behaviors
- Select an object.
- In the properties panel, click on "Add a Behavior".
- Choose the behavior that fits your needs (e.g., platformer, physics).

## Running the Game
- Press the play button to run the game.

## Interacting with the Game
- Use keyboard or mouse controls to interact with your objects.

## Returning to the Editor
- Close the game preview to return to the GDevelop editor.

## Complete Exercise with Controllable Square
- Create a sprite of a square.
- Add platformer behavior to the square object.
- Test the movement controls.

## Tips
- Always save your project frequently.
- Use the preview function to test changes instantly.
