# GDevelop Installation and Setup

## Objectives
- Understand what GDevelop is and its applications.
- Learn the system requirements for running GDevelop.
- Follow installation steps for different operating systems.
- Configure GDevelop for initial use.
- Complete a practical exercise to familiarize yourself with the software.
- Discover additional resources for learning GDevelop.

## What is GDevelop?
GDevelop is an open-source game development platform designed for everyone, from beginners to experienced developers. It enables users to create games without coding, utilizing a visual programming interface and offering a wide range of features and integrations.

## System Requirements
### Windows
- OS: Windows 7 or later
- CPU: Intel or AMD processor with 1 GHz or faster
- RAM: 2 GB or more
- Graphics: OpenGL 2.0 or later compatible graphics card
- Disk Space: At least 300 MB free

### macOS
- OS: macOS 10.12 or later
- CPU: Intel processor or Apple Silicon
- RAM: 2 GB or more
- Graphics: OpenGL 2.0 compatible graphics
- Disk Space: At least 300 MB free

### Linux
- OS: Ubuntu 18.04 or later, or other distributions with x64 support
- CPU: Intel or AMD processor with 1 GHz or faster
- RAM: 2 GB or more
- Graphics: OpenGL 2.0 compatible graphics
- Disk Space: At least 300 MB free

## Installation Steps
### Windows
1. Visit the [GDevelop website](https://gdevelop-app.com/).
2. Go to the Download section and choose the Windows installer.
3. Run the installer and follow the on-screen instructions.
4. Launch GDevelop from the Start menu.

### macOS
1. Go to the [GDevelop website](https://gdevelop-app.com/).
2. Download the macOS version from the Downloads section.
3. Drag and drop GDevelop into the Applications folder.
4. Start GDevelop from the Applications folder.

### Linux
1. Access the [GDevelop website](https://gdevelop-app.com/).
2. Download the AppImage file or install via your distribution's package manager.
3. If using AppImage, make it executable with: `chmod +x GDevelop-x.y.z.AppImage`.
4. Run the AppImage by double-clicking or from terminal.

## Initial Configuration
After installation:
- Launch GDevelop and create a new project.
- Explore the interface and familiarize yourself with tools available.
- Set your preferences in the Settings menu.

## Practical Exercise
1. Create a simple game using GDevelop by following the built-in tutorials.
2. Experiment with adding sprites and creating scenes.
3. Test your game in preview mode to see your changes in action.
