# Estrutura dos Módulos do GDevelop Academy

A estrutura de módulos para o GDevelop Academy segue um formato padronizado que facilita o aprendizado e a compreensão dos conceitos.

## Tópicos Abordados

1. **Introdução ao GDevelop**  
   - O que é GDevelop?  
   - Como instalar o software?  

2. **Criando seu Primeiro Jogo**  
   - Passo a passo para iniciar um projeto.  
   - Configuração do ambiente de desenvolvimento.  

3. **Elementos do Jogo**  
   - Sprites e animações  
   - Sons e efeitos  

4. **Lógica do Jogo**  
   - Eventos e ações  
   - Variáveis e estados

5. **Publicação e Divulgação**  
   - Como exportar jogos  
   - Melhorando a visibilidade do jogo

## Conclusão

O GDevelop Academy visa fornecer uma base sólida para novos desenvolvedores, permitindo que eles se tornem proficientes na criação de jogos utilizando a plataforma GDevelop.
